Linux Programming (GCC, Make, GDB)

GCC

A compiler turns human-readable source code into machine-readable object code that can actually run.

The compilers of choice on Linux systems are all part of the GNU Compiler Collection, usually known as GCC. GCC include compilers for C, C++, Java, Objective-C, Fortran, and Chill.

The name of the C compiler is gcc and the name of the C++ compiler is g++.

Compiling a Single Source File

To compile a C source file using gcc, the -c option is used:

gcc -c main.c

To compile a C++ source file using g++, the -c option is used:

g++ -c main.c

The -c option tells gcc and g++ to compile the program to an object file only; without it, gcc and g++ will attempt to link the program to produce an executable. So the result after previous commands is an object file called main.o.

By default, GCC searchs for header files in the current directory and in the directories where headers for the standard libraries are installed. To include header files from somewhere else, the -I option is needed:

g++ -c -I ../include main.c

Others (Man, Info)

Man Pages

The man pages are divided into numbered sections; for programmers, the most important are:

  • (1) User commands
  • (2) System calls
  • (3) Standard library functions
  • (8) System/administrative commands

To look up a man page of a "name" command or function: 

man name

For example, to get the man page for the sleep command (in section 1 of the Linux man pages): man sleep. To see the man page for the sleep library function, use the command: man 3 sleep

Displays all man pages (in all sections) for a command or function matching "name":

whatis name

Perform a keyword search on the summary lines:

man -k keyword

Info

The Info documentation system contains more detailed documentation for many core components of the GNU/Linux system, plus several other programs. Info pages are hypertext documents.

To launch the text-based Info browser, just type info in a shell window.

To display the keys for navigating an Info document: ctrl + H

To jump directly to a particular Info document by specifying the page name on the command line: info pageName

In Emacs, you can access the built-in Info browser by typing M-x info or C-h i
