Learning from Data

The learning problem

The essence of machine learning has tree componentes:

  • A pattern exists: If a pattern didn't exist, there would be nothing to look for
  • The problem cannot be pinned down mathematically
  • There are data in the problem

If these three componentes are found in a problem in some field, then machine learning can be an application tool.

Components of learning:

  • Input: x. In can be seen as a d-dimensional vector.
  • Output: y
  • Target function: ƒ: X → Y
  • Data: (x1,y1), (x2,y2), ..., (xN,yN)
  • Hypothesis: ƒ: X → Y. Is the "formula" that is obtained to approximate the target function.
